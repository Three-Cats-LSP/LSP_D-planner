# 🎨 LSP D-PLANNER BETA 3.6 — RELEASE NOTES

**Version:** Beta 3.6  
**Date:** May 26, 2026  
**Focus:** Design Polish + Sharp Graphs  

---

## What's New

### Full Design Overhaul
Every UI component has been refined — tighter spacing, better typography hierarchy, cleaner card titles, improved buttons and stats. The app feels noticeably more polished on both desktop and mobile.

### Sharp Graphs (HiDPI / Retina)
Canvas graphs now render at full screen resolution. On phones with high-density screens the text is crisp and clear instead of blurry.

### Numbered Stop Dots
Every dot on the dive profile and GF curve now has a number inside it, making it easy to cross-reference with the legend below.

### Mobile Graph Legends
On mobile screens, graph labels are replaced with a clean numbered table below each graph:
- **Dive Profile:** Stop · Run time · ppO₂
- **GF Curve:** Stop · Gradient Factor %

### Bühlmann Tab Order
Deco Schedule is now the first tab and default view in Bühlmann mode. The full tab order is:
`Deco Schedule | Dive Planner | Multi Dive | CNS O₂ | NDL Tables | Reference`

### Smarter PDF Layout
3 pages instead of 4:
- **Page 1:** Dive Profile Table
- **Page 2:** Dive Profile Graph + GF Curve
- **Page 3:** Tissue Saturation
