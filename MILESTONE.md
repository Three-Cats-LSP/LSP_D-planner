### Text Export (`exportDecoSchedule`, `exportContingencyTXT`)
- **Header cleanup:** "DECOMPRESSION SCHEDULE" (removed "DIVE PLANNER -")
- **Date/time relocation:** Moved from beginning to end
- **Footer format:** "Generated by LSP D-PLANNER: [localeString]"
- **Contingency header:** "CONTINGENCY PLAN" (removed "LSP D-PLANNER —")
- **Filename format:** Scenario-aware naming (Standard_Plan, Lost_*, Extra_*, Combined)

### PDF Export (`exportPDF`, `exportContingencyPDF`)
- **Filename via title tag:** Uses scenario naming format
- **Main dive:** `dive-plan-YYYY-MM-DD_Standard_Plan`
- **Contingency:** Uses same logic as text files
- **Window behavior:** Removed size parameters, opens in tab not window
- **ISO date:** Added `isoDate` variable for consistent YYYY-MM-DD format

### Contingency Scenario System
- **Auto-recalc triggers:** Added oninput handlers to:
  - `decoDepth` (depth input)
  - `decoBT` (bottom time input)
  - `decoGas` (gas selection)
  - `decoCustomO2` (custom O₂ input)
  - `setCustomGF()` function (GF Low/High changes)
- **Scenario labels:** Changed "Without" to "Lost" throughout
- **Filename logic:** Dynamically builds scenario names from `contGasLose` and `contExtraBT`

### Mode Switching (`setAlgo`)
- **Body class added:** `document.body.classList.toggle('algo-buh', isBuh)`
- **CSS selector:** `.buh-only { display: none !important; }` and `body.algo-buh .buh-only { display: flex !important; }`
- **Tab visibility:** Deco Schedule tab now shows/hides correctly

### Mobile Clipboard (`copyFallback`)
- **New function:** Dedicated `copyFallback()` for non-Clipboard API environments
- **Error handling:** Try-catch with console logging
- **Positioning:** `left: -9999px; top: -9999px;` for better offscreen placement
- **Focus:** Explicit `focus()` before `select()`
- **Success check:** Validates `execCommand('copy')` result
