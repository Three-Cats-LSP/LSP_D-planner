# Security Policy

## Security Considerations

LSP D-PLANNER is designed with security and safety in mind.

### Security Features
- No external dependencies
- No API calls or cloud services
- All data stored locally
- Offline operation
- Open source (fully auditable)
- No tracking or analytics

### Reporting Security Issues

**DO NOT** open public issues for security vulnerabilities.

Please email security concerns privately to [your-email]

Include:
- Description of vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix if available

### Safety Disclaimer

This tool is for training and planning purposes only:
- NOT a dive computer replacement
- Should NOT be used as sole planning source
- Always use professional dive computers
- Never dive exact table limits
- Follow proper training and certification
