# Installation Guide for LSP D-PLANNER

## Quick Start (Recommended)

1. **Download** `index.html` from releases
2. **Open** in web browser
3. **Use** immediately - no installation needed!

## Installation Methods

### Method 1: Direct Browser
- Download index.html
- Double-click to open
- Works immediately

### Method 2: Web Server
```bash
# Python 3
python3 -m http.server 8000

# Then visit: http://localhost:8000/index.html
```

### Method 3: Local File
- Save to Documents, Desktop, or Downloads
- Right-click → Open with Browser
- Bookmark for quick access

### Method 4: PWA (Mobile)
1. Open index.html in mobile browser
2. Tap menu → "Add to Home Screen"
3. App appears as installable app
4. Works offline like native app

## System Requirements

- **Browsers**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **Mobile**: iOS 14+ or Android 11+
- **Storage**: 250 KB
- **Internet**: Only needed for download
- **Offline**: Fully functional offline

## Troubleshooting

### Not loading?
- Check browser compatibility
- Try a different browser
- Clear browser cache
- Download fresh copy

### Copy not working?
- Check browser permissions
- Try manual copy (Ctrl+C)
- Update browser to latest version

### Mobile not working?
- Update to latest iOS/Android
- Try different browser
- Clear app cache
- Reinstall from home screen

### Export issues?
- Check popup blocker
- Ensure sufficient storage
- Try different export format
- Download fresh copy
