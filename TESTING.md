# Testing Guide for LSP D-PLANNER

## Manual Testing Checklist

### Core Features
- [ ] REC mode calculations work
- [ ] Bühlmann mode calculations work
- [ ] Tools mode functions correctly
- [ ] NDL table displays correctly
- [ ] GF settings apply properly

### Export Functions
- [ ] COPY to clipboard works
- [ ] TXT export downloads
- [ ] PDF export generates
- [ ] Filenames are correct
- [ ] Content is formatted properly

### Contingency Planning
- [ ] Emergency plan calculates
- [ ] Gas loss scenarios work
- [ ] Extra BT scenarios work
- [ ] Combined scenarios work
- [ ] Auto-recalculation functions

### Mobile
- [ ] Works on iOS Safari
- [ ] Works on Android Chrome
- [ ] Responsive design works
- [ ] Touch controls work
- [ ] Copy works on mobile
- [ ] PWA installable

### Browsers
- [ ] Chrome desktop
- [ ] Firefox desktop
- [ ] Safari desktop
- [ ] Edge desktop
- [ ] iOS Safari
- [ ] Android Chrome

### Offline
- [ ] Works without internet
- [ ] All features available offline
- [ ] Exports work offline
- [ ] PWA functions offline

## Performance Testing

- Load time < 2 seconds
- Memory usage < 20MB
- Smooth scrolling
- Fast calculations
- Quick exports

## Safety Testing

- Disclaimer visible
- Warnings displayed
- Conservative defaults
- Clear instructions
- Proper disclaimers on exports

## Bug Reporting

Include:
- Browser and version
- OS and version
- Device type
- Steps to reproduce
- Expected vs actual
- Screenshots if applicable
