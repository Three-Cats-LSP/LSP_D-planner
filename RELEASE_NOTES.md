# LSP D-PLANNER BETA 5.6.3 — Release Summary

## 🎯 What's New?

This release focuses on **export consistency**, **mobile compatibility**, and **user experience improvements**.

### Key Highlights

✨ **Unified Export Filenames**
- All exports now use consistent naming: `dive-plan-YYYY-MM-DD_[scenario].txt|pdf`
- Easy to organize and identify dive plans by scenario
- Standard Plan, Gas Loss, Extra BT, and Combined scenarios clearly labeled

🔧 **Mobile Copy Fixed**
- COPY button now works reliably on iOS Safari, Android Chrome, and all mobile browsers
- Improved clipboard fallback for non-HTTPS environments
- No more silent failures on mobile devices

📝 **Clean Export Headers**
- Removed redundant "LSP D-PLANNER —" prefixes
- Date/time moved to end for better readability
- Professional format: "Generated by LSP D-PLANNER: [date time]"

🚨 **Contingency Plan Enhancements**
- Auto-recalculation when dive parameters change
- "Lost" terminology instead of "Without" (more intuitive)
- Real-time scenario updates without manual button clicks

📊 **GF Values Now Display Correctly**
- Fixed bug where GF showed "—/—" in copy output
- Now shows actual values: "GF: 30/70", "GF: 40/85", etc.

🗂️ **PDF Export Improvements**
- Both standard and contingency PDFs now open in new tabs (consistent behavior)
- Filename visible when saving to disk
- No forced window resizing
