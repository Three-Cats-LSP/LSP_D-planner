# Troubleshooting Guide

## Common Issues

### App Won't Load
**Problem**: Blank screen or error message
**Solutions**:
1. Clear browser cache (Ctrl+Shift+Del)
2. Try a different browser
3. Download fresh copy of index.html
4. Check if JavaScript is enabled

### COPY Button Not Working
**Problem**: Nothing copied to clipboard
**Solutions**:
1. Check browser permissions for clipboard
2. Allow pop-ups in browser settings
3. Try manual copy (highlight all + Ctrl+C)
4. Update browser to latest version
5. Try different browser

### Export Not Working
**Problem**: TXT/PDF not downloading
**Solutions**:
1. Disable pop-up blockers
2. Check Download folder
3. Try different browser
4. Check available storage space
5. Try different export format

### Calculations Seem Wrong
**Problem**: Results don't match expectations
**Solutions**:
1. Verify input values are correct
2. Check water density setting
3. Verify algorithm selection (REC vs Bühlmann)
4. Check GF settings (Bühlmann mode)
5. Compare with dive computer result

### Mobile Issues
**Problem**: App not working on phone
**Solutions**:
1. Update iOS/Android to latest version
2. Try different mobile browser
3. Clear browser cache
4. Reinstall from home screen
5. Check available storage space

### Slow Performance
**Problem**: App running slowly
**Solutions**:
1. Close other browser tabs
2. Clear browser cache
3. Restart browser
4. Check available RAM
5. Try different browser

## Browser-Specific Issues

### Safari (iOS/macOS)
- Enable JavaScript in settings
- Allow clipboard access
- Check pop-up blocker
- Update to latest version

### Chrome (Android)
- Clear app cache
- Check storage space
- Enable JavaScript
- Allow clipboard permission

### Firefox
- Disable privacy mode
- Check add-ons/extensions
- Clear cache
- Update to latest version

### Edge/IE
- Update to Edge (IE not supported)
- Enable JavaScript
- Clear cache
- Check security settings

## Still Not Working?

1. Note exact error message
2. Check browser console (F12)
3. Try incognito/private mode
4. Download fresh copy
5. Open GitHub issue with details

## Safety Reminders

⚠️ **IMPORTANT**: If something seems wrong:
- STOP and abort the dive
- Do NOT use calculations without verification
- Use dive computer for actual diving
- Consult with certified dive master
- Err on side of caution always
